# Mahboob ur Rahman — Portfolio

Personal portfolio website. Cyber Security · Cloud · DevOps · AI Operations · Technology Trainer.

**Stack:** Single-file static site (HTML + CSS + vanilla JS). Zero build step.

## Run locally
Just open `index.html` in a browser, or serve it:
```bash
python3 -m http.server 8000
# visit http://localhost:8000
```

## Deploy
This is a static site — it deploys as-is on Vercel, Netlify, GitHub Pages, or any static host.
No build command, no framework. `index.html` is the entry point.

## Files
- `index.html` — the site
- `Mahboob_ur_Rahman_CV.pdf` — downloadable CV (linked from the site)
